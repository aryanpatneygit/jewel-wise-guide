| #  | Data Category                     | Description / What Exactly Is Tracked                                                       |
|----|----------------------------------|-----------------------------------------------------------------------------------------------|
| 1  | Product ID / SKU                 | Unique alphanumeric identifier assigned to the bracelet.                                     |
| 2  | Bracelet Type                    | Tennis Bracelet, Charm Bracelet, Cuff Bracelet, Bangle Bracelet, Link Bracelet, Kada.        |
| 3  | Primary Metal                    | Gold, Silver, Platinum, Rose Gold, White Gold, Titanium.                                     |

| 6  | Net Metal Weight                 | Pure metal weight excluding stones (in grams).                                               |
| 7  | Stone Type                       | Diamond, Ruby, Emerald, Sapphire, Moissanite, Cubic Zirconia, Polki, Kundan.                 |
| 8  | Stone Count                      | Exact number of stones present in the bracelet.                                              |
| 9  | Total Stone Carat Weight         | Combined carat weight of all stones used.                                                    |
| 10 | Stone Quality                    | For diamonds: Cut (Round/Princess/Oval), Clarity (VVS1/VS1/SI1), Color (D–J).                |
| 11 | Stone Setting Style              | Prong Setting, Bezel Setting, Channel Setting, Pave Setting, Invisible Setting.              |
| 12 | Stone Certification Number       | IGI Certificate ID, GIA Certificate ID, or SGL Certificate ID (when applicable).              |
| 13 | Hallmark Details                 | BIS Hallmark ID, Year of hallmarking, Assaying center code.                                  |
| 14 | Bracelet Length                  | Length of bracelet in inches or centimeters (e.g., 6.5 in, 7.0 in, 7.5 in).                  |
| 15 | Bracelet Size Category           | Small, Medium, Large, Adjustable.                                                            |
| 16 | Clasp / Lock Type                | Box Clasp, Lobster Clasp, S-Hook, Magnetic Clasp, Toggle Clasp, Screw Lock.                  |
| 17 | Surface Finish                   | High Polish, Matte Finish, Sandblasted, Hammered Finish, Rhodium-plated Finish.              |
| 18 | Design Style                     | Modern Minimalist, Antique, Temple Design, Floral Design, Geometric Pattern, Heritage.       |
| 19 | Chain/Link Style (if applicable) | Cuban Link, Rope Chain, Figaro Link, Snake Chain, Box Chain, Curb Link.                      |
| 20 | Making Charges                   | Crafting cost in percentage or fixed amount.                                                  |
| 21 | Wastage Percentage               | Percentage of additional metal used during manufacturing.                                     |
| 22 | Pricing Breakdown                | Gold Value, Stone Value, Making Charges, Wastage Value, GST.                                 |
| 23 | Final Selling Price              | Final retail price after all cost components.                                                 |
| 24 | Supplier / Vendor Name           | Vendor or artisan who manufactured the bracelet or supplied materials.                        |
| 25 | Stock Location                   | Showroom A, Warehouse B, Safe Locker 1, Repair Desk, Branch 2.                                |
| 26 | Availability Status              | In Stock, Out of Stock, Made-to-Order, Reserved, Under Repair.                                |
| 27 | Manufacturing Date               | Date on which the bracelet production was completed.                                          |
| 28 | Lead Time for Reproduction       | Number of days required to reproduce the bracelet design.                                     |
| 29 | Media Assets                     | Image URLs, 360° video URL, CAD file name for the bracelet.                                   |
| 30 | Product Lifecycle Stage          | New Arrival, Fast-moving, Slow-moving, Ageing Stock, Discontinued.                            |
